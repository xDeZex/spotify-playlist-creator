# Album Playlists created oldest-release-date first

Album Playlists for a given artist are created in ascending release-date order (oldest first). This means the newest album is created last and therefore sits at the top of Spotify's "recently added" view — the natural sort order a user expects when browsing the library.

The previous behaviour created newest-first, which caused the oldest album to appear at the top of "recently added" — the opposite of what users want.
